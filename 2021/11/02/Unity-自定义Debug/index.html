啥？你不会搞Unity中可以用的类库？

  1. VS打开Unity工程，在解决方案资源管理器->解决方案目录下右击->添加->新建项目
  2. 选择类库(.Net Framework)进行创建。
  3. 右击创建的类库->属性->用程序->目标框架选择Unity 3.5 .net Subset Base Class Libraries，保存即可。(要是没这个选项就是你创建的类库类型不对，回第二部重新来过)
